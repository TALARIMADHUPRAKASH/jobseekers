Steps For Installation  and execution of project:

1.Install Xampp 7.2.4 in Your Desktop or Laptop.

2.copy the main folder in the project to C:\xampp\htdocs

2.open xampp control panel and start apache and mysql services.

3.click on admin button beside mysql it will open your default browser and shows phpmyadmin page for database creation and etc.

4.click on import tab on the top and click on choose file and import 127_0_0_1 .sql file

5.Now open Your browser(prefer Google chrome) and type/paste http://localhost/main/home.html
